# Code of Conduct

This project adopts the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — issues, pull requests, discussions, or
any other channel — you agree to abide by its terms.

## Scope

This Code of Conduct applies within all project spaces — the repository,
issues, pull requests, discussions, and any other official channel — and
also when an individual is representing the project in public spaces.

## Reporting

Instances of abusive, harassing, or otherwise unacceptable behavior may be
reported by emailing
[conduct@agentidentitytrustprotocol.org](mailto:conduct@agentidentitytrustprotocol.org).
All complaints will be reviewed and investigated promptly and fairly. The
reporter's privacy and security will be respected.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code
of Conduct and will take appropriate, fair corrective action in response to
any behavior they deem inappropriate, threatening, offensive, or harmful.
The enforcement guidelines are those of
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines).
